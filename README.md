# Simple Calculator

A basic command-line calculator built with Python that performs arithmetic operations including addition, subtraction, multiplication, and division.

## Features

- Addition
- Subtraction
- Multiplication
- Division
- Input validation
- Division-by-zero handling
- Interactive menu system

## Requirements

- Python 3.6 or later

## Installation

1. Clone the repository:

```bash
git clone https://github.com/your-username/simple-calculator.git
```

2. Navigate to the project folder:

```bash
cd simple-calculator
```

3. Run the application:

```bash
python calculator.py
```

## Usage

Example:

```text
=== Simple Calculator ===

Operations:
1. Addition
2. Subtraction
3. Multiplication
4. Division
5. Exit

Select an operation (1-5): 1
Enter first number: 10
Enter second number: 5

10.0 + 5.0 = 15.0
```

## Functions

| Function | Description |
|-----------|-------------|
| add(a, b) | Returns the sum of two numbers |
| subtract(a, b) | Returns the difference of two numbers |
| multiply(a, b) | Returns the product of two numbers |
| divide(a, b) | Returns the quotient or an error message |
